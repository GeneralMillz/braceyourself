

This folder is configured for GitHub Pages deployment to `braceyourself.evergreenmediadesign.com`.

### Files
- `CNAME` - Contains: `braceyourself.evergreenmediadesign.com`
- `index.html` - Main landing page
- `README.md` - This file

### Deployment

1. Create a new GitHub repository: `braceyourself`
2. Copy this folder contents to the repo root
3. Push to GitHub
4. Go to repo **Settings → Pages**
5. Enable GitHub Pages from main branch
6. Add custom domain: `braceyourself.evergreenmediadesign.com`
7. GitHub will auto-detect CNAME and provision SSL

### DNS Configuration

Add this CNAME record in Squarespace Domains:

| Host | Type | Value | TTL |
|------|------|-------|-----|
| `braceyourself` | CNAME | `generalmillz.github.io` | Auto |
