# Roadmap

## Phase 1 — Stabilization
- Ensure 100% functional parity with legacy version.
- Validate template loading and backward compatibility.
- Improve error handling and fallback logic.

## Phase 2 — Performance
- Add dirty-cell rendering.
- Optimize image quantization.
- Improve mobile performance.

## Phase 3 — UX Enhancements
- Expand guided tour.
- Improve mobile palette interactions.
- Add more intuitive color selection tools.

## Phase 4 — Extensibility
- Add automated tests.
- Add CI/CD pipeline.
- Prepare for template format migration.
- Add optional analytics and feedback tools.
