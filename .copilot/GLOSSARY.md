# Glossary

**Grid**  
The 2D canvas where the bracelet pattern is edited.

**Cell**  
A single square in the grid representing one stitch.

**Template**  
A predefined JSON pattern that can be loaded into the grid.

**Legacy Template**  
Older template format requiring compatibility handling.

**Palette**  
The set of colors available for the current design.

**Quantization**  
Reducing an imported image to a limited color palette.

**Guided Mode**  
An optional onboarding flow that introduces features.

**Export**  
Generating a downloadable or copyable representation of the pattern.
